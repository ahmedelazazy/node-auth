# Node Auth App

NodeJS Passport Authentication Boilerplate

## Installation

Use the package manager [npm] to install.

```bash
npm install --save
```

## Features

* Registeration
* Login
* Logout
* Members only access
* Password hashing
* Validations
* Flash messages

## Used Packages

* Express
* Mongoose
* Passport
* Connect-flash
* Express-validator
* Jade
* Bcrypt
* Multer
